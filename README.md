# Dad-Jokes-App

This is a multi-page web app I built using React that allows the user to click a button on the webpage to display a random Joke.

### To Install

```
git clone https://github.com/Ben-893/Dad-Jokes-React && cd Dad-Jokes-React
yarn install
```

### To Use

```
- yarn install
- yarn start
```

### Technologies used

- React
- JavaScript
- HTML
- CSS
